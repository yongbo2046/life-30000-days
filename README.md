# 人生三万天

一个简洁优雅的网页应用，用于可视化展示人生的三万天。

## 功能特点

- 📊 **网格可视化** - 用方块展示人生进度，已过的时间用紫色标记
- 🔄 **双视图模式** - 支持按周展示（4286周）和按日展示（30000天）
- ⏱️ **实时倒计时** - 精确显示距离30000天还有多少天、小时、分钟、秒
- 📈 **统计数据** - 显示已度过天数、剩余天数、人生进度百分比
- 💾 **数据持久化** - 生日信息自动保存，下次打开自动加载
- 📱 **响应式设计** - 完美支持手机和电脑浏览

## 在线访问

🌐 **https://yongbo2046.github.io/life-30000-days/**

## 使用方法

1. 访问在线版本或在浏览器中打开 `index.html`
2. 输入你的出生日期
3. 点击"计算"按钮
4. 查看你的人生进度和倒计时
5. 使用右上角的切换按钮在"按周展示"和"按日展示"之间切换

## 技术栈

- 纯原生 HTML/CSS/JavaScript
- 无需任何依赖
- 可直接在浏览器中运行
